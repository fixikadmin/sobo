Lucram cu tmux

Instalarea tmux

In Debian

apt-get install tmux

In Centos

yum install tmux

Startare tmux si lucrul cu ecranele

Tmux se starteaza prin comanda:

tmux

Pentru a deschide ecrane noi in tmux folositi combinatia de taste CTRL + b si apoi separat %. Pentru a tranzita intre ferestre folositi combinatiile de taste CTRL + b urmat de o sau CTRL + b urmat de >, <, ^, v.

Pentru a inchide un ecran inchideti shell-ul folosind CTRL + d sau exit sau logout.

Pentru a va detasa de tmux folositi combinatia de taste ctrl + b urmat de d.

Pentru a va reatasa la tmux folositi comanda 

tmux attach

Alte comenzi:

  1. Inchide panelul curent: CTRL - b + x

  2. Adauga un panel in fereastra curenta (pe oriozontala): CTRL - b + SHIFT - ”

  3. Interschima continutul intre peneluri: CTRL - b + SHIFT - [ si CTRL - b + SHIFT - ]

  4. Listeaza sesiunile tmux: CTRL - b + w

  5. Arata numarul panelulului: CTRL - b + q

  6. Trece la urmatoarea sesiune tmux: CTRL - b + n

  7. Trece la sesiune tmux precedenta: CTRL - b + p

  8. Arata toate comenzile posibile: tmux list-keys
